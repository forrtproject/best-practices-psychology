# Best-Practice Guidelines for Psychology

A community project, supported by [FORRT](https://forrt.org), to develop **consensus-based best-practice guidelines** that operationalize the &ldquo;Certified&rdquo; level of the [TOP Guidelines](https://www.cos.io/initiatives/top-guidelines) for psychological research.

This repository hosts the project's standalone landing page, deployed via GitHub Pages.

## Local preview

Open `index.html` in a browser, or serve the directory:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
